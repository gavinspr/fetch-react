# Fetch A Friend

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (v16.14.0 or higher)
- pnpm (v8.0.0 or higher)
  ```bash
  npm install -g pnpm

  ```

## Install

```bash
git clone https://github.com/gavinspr/fetch-react.git
```

```bash
cd fetch-react
```

```bash
pnpm install
```

```bash
pnpm dev
```
